# Vite bundled dev with rolldown

Integrating [Rolldown HMR](https://github.com/rolldown/rolldown/tree/hmr-poc) as Vite plugin.

```sh
pnpm dev
pnpm -C examples/ssr dev
pnpm -C examples/react dev
pnpm -C examples/mpa dev
```
